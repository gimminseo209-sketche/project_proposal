# project_proposal

로또 번호 생성기
: 랜덤으로 1~45 사이 숫자 6개 출력하고, 중복 없게 만드는 프로그램.

Lotto Number Generator(로또 번호 생성기)
1. 프로젝트 기획 의도

사용자가 로또 번호를 무작위로 생성할 수 있도록 돕는 간단한 프로그램입니다.
로또 번호 추첨의 핵심 조건인 중복 없는 숫자 선택과 범위 제한(1~45)을 코드로 구현하여,
파이썬의 기본 라이브러리와 명령줄 인자 처리 기능을 연습하는 것을 목표로 하였습니다.

또한 단순한 난수 생성이 아닌,
사용자가 숫자 개수, 범위, 랜덤 시드를 직접 설정할 수 있도록 설계하여
프로그램의 확장성과 재사용성을 고려하였습니다.

2. 프로그램 기능 설명

본 프로그램은 다음과 같은 기능을 제공합니다.
1. 1부터 45 사이의 숫자 중 중복 없이 6개 숫자 생성
2. 생성된 숫자를 오름차순으로 정렬하여 출력
3. 명령줄 옵션을 통해 기능 조절 가능

-n / --count : 뽑을 숫자의 개수 설정

--min : 최소 숫자 설정

--max : 최대 숫자 설정

--seed : 랜덤 시드 설정 (재현 가능한 결과)

이를 통해 사용자는 기본 로또 번호 생성뿐 아니라,
다양한 조건의 번호 조합을 실험해볼 수 있습니다.

3. 구현 방법 개요
프로그램은 파이썬의 표준 라이브러리만을 사용하여 구현되었습니다.

random.sample()
→ 중복 없는 난수 생성을 위해 사용

argparse
→ 명령줄 인자를 처리하여 사용자 입력을 유연하게 받도록 설계

함수 단위 설계
→ 번호 생성 로직(generate_lotto_numbers)과 실행 로직(main)을 분리하여 가독성을 높임

또한 Juypter 및 VS Code 환경에서도 오류 없이 실행되도록
parse_known_args()를 사용하여 실행 환경에 대한 호환성을 고려하였습니다.

4. 패키지 구조
```
my_package
├─ __init__.py
└─ __main__.py
```
my_package는 프로그램 전체를 하나의 패키지로 구성하기 위한 디렉터리이며,
__init__.py는 해당 디렉터리를 파이썬 패키지로 인식하게 합니다.
__main__.py는 패키지 실행 시 자동으로 실행되는 파일로,
프로그램의 주요 로직과 실행 기능을 담당합니다.
